About Smart Waste Management

This project monitors campus trash bins with ultrasonic sensors and displays real-time fill levels. It supports up to 10 bins, with initial testing on 1 bin. Monitoring is active daily between 06:00 and 22:00.

Key Features

  • Real-time bin fill-level monitoring
  • Alerts for overfilled bins
  • Cleaner dashboard for task management

Technology Stack

  • Backend: Spring Boot (Java)
  • Frontend: Angular 19
  • Devices: Arduino / Raspberry Pi with ultrasonic sensors
